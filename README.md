# Dog Breed Classification API – Early Project

This project is a Python-based API built with Flask that uses a Convolutional Neural Network (CNN) to identify a dog's breed from an image.

It demonstrates a complete end-to-end workflow, including:

- Image preprocessing
- Breed classification using a trained CNN model
- Serving predictions via a RESTful API

This project represents foundational work in AI and machine learning, showcasing my ability to design and deploy working models.  
My current projects build on these skills, creating more advanced, production-ready AI systems that are containerized and capable of handling complex real-world tasks (not yet publicly released).

## Features

- Easy-to-use API endpoint for dog breed prediction
- Fast inference for individual images
- Clear and modular Python code for educational and demonstration purposes

## Getting Started

1. Clone the repository  
2. Install the required dependencies (`pip install -r requirements.txt`)  
3. Run the Flask API (`python app.py`)  
4. Send a POST request with an image to the `/predict` endpoint to get the breed prediction
